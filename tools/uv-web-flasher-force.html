UV-K5 / K6 浏览器刷机

界面灵感来自 UVTools，协议来自 uvmod。本页不进行 CRC / 机型版本校验，仅作个人风险下的强制写入。

错误镜像可能损坏对讲机。请先关机，按住 PTT 再开机进入刷机模式（手电常亮），再插 USB；使用 Chrome / Edge。
握手盲写：不等待引导发来的 0x18，只发版本包后延时即写入。
块盲写：若勾选「不等块应答」，写入时完全不看串口返回（含每块 0x1a），仅连续发数据；易写坏或 USB 缓冲溢出，请自担风险。

支持 firmware.packed.bin（打包格式）或写频软件用的原始 firmware.bin（最大 61440 / 0xF000 字节）。
